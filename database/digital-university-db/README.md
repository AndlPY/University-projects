# Digital University Database

Digital University Database is an academic PostgreSQL project that models the core data structure of a modern university. 

**The repository contains:** 
- materials for domain analysis
- relational schema design 
- database creation scripts
- seed data
- SQL queries prepared for a sequence of laboratory assignments.

The database covers the main university entities.
The project is designed to be used with PostgreSQL and DBeaver, with special attention to clear schema organization, referential integrity, and practical SQL examples for data definition, insertion, modification, deletion, and reporting.
